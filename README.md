# CryptoDiary
> author : Yopiyama

## 説明

日記暗号化，復号アプリです．日記以外の用途でも使えます．

GUIが基本部分はほぼ完成です．ただ裏の処理の実装がまだ完成はしていないです．それらがおわったら暗号方式を順次追加していこうと思っています．．

> 今の状態ではRSAの鍵長が1024bitとなっていますがこれはデバッグの際に鍵生成に時間が多くかかると不便だから意図的に短くしています．
